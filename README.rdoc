== Exercise

This exercise is a sample application that implements JSON remote calls to iTunes store. It is built on Twitter Bootstrap and supports mobile platforms.

Demo is available here: http://espark.herokuapp.com

== Getting started

The application does not use DB, so you should run only:

#bundle

== Tests

Run:

#cucumber

#rspec

== Launch

Run:

#rails s

Then in the browser:

http://localhost:3000/
